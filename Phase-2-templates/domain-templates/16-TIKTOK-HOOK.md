```

REQUIREMENTS:
- HOOK OPTIONS (5 variations): Each under 5 seconds, different angles on same topic
- SCRIPT STRUCTURE:
  * Second 0-3: Visual + audio hook that stops scroll
  * Second 3-15: Problem setup or intrigue build
  * Second 15-45: Solution/reveal/punchline with value
  * Second 45-60: CTA or engagement trigger

- CAPTION STRATEGY: On-screen text placement and timing

- Script must work WITHOUT sound (captions essential)

- [Timestamp]: [Text]

**CAPTION/CTA**:

**HASHTAGS**: #[Specific] #[Niche] #[Broader] #[Trending] #[Community]
```

## 🔧 VARIATIONS

**Variation 1: Trend Participation**
```
Hook: Use current trending sound with your twist
Structure: Setup (3 sec) → Trend moment (15 sec) → Punchline (15 sec) → CTA (5 sec)
Key: Get to the trend audio within 3 seconds
```

**Variation 2: Duet/Stitch Response**
```
Hook: Start with original video's premise, then pivot
Structure: Original context (5 sec) → Your take/reaction (20 sec) → Counterpoint (20 sec) → CTA (5 sec)
Engagement: Ask viewers if they agree with your take
```

**Variation 3: Transformation/Before-After**
```
Hook: "Day 1 of [Challenge]" or "The [Timeframe] transformation nobody expected"
Structure: Setup → Process montage → Result reveal → Reaction → CTA
Visual: Heavy use of trending transition sounds
```

## ✅ QUALITY CHECKLIST

- [ ] Hook happens in first 3 seconds (not 10 seconds of intro)
- [ ] Visual and audio are synced and reinforcing
- [ ] Text overlays are under 20 words per screen
- [ ] Script works with and without sound
- [ ] CTA is specific and easy to complete
- [ ] Hashtags mix niche-specific and broader discovery tags
- [ ] Video has natural loop point for replays
- [ ] Every 5 seconds has visual change or pace shift

---

## 💡 PRO TIPS

- **The pattern interrupt**: Use unexpected visual or audio in first frame
- **Caption everything**: 80%+ watch without sound
- **The 3-second test**: If they don't understand the premise by second 3, rewrite
- **One idea per video**: Don't explain your entire system — give one golden nugget
- **Text as visual**: Move text around, use different sizes/colors for emphasis
- **The cliffhanger ending**: "Part 2" can drive follows, but deliver on the promise

## 🔄 ITERATION WORKFLOW

1. **Hook testing**: Create 5 versions of opening, test with small audience
2. **Pace check**: Watch at 2x speed — if it's too fast, it's just right
3. **Text timing**: Ensure on-screen text stays up long enough to read
4. **Sound-off test**: Watch without audio — does it still make sense?
5. **Post analysis**: Check drop-off points, note where viewers leave
6. **Hook library**: Save high-performing hooks, remix for new content
