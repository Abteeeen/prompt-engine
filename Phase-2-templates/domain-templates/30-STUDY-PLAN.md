## 🔧 VARIATIONS

**Variation 1: Cram Schedule (1-2 Weeks)**
```
Focus: High-yield topics only, emergency protocol
Strategy:
• 80/20 analysis: What 20% of content drives 80% of points?
• Active recall only — no passive reading
• Practice tests as primary learning tool
• Sleep prioritized over extra study hours
Realistic expectation: Damage control, not optimal learning
```

**Variation 2: Long-Term Skill Building (6+ Months)**
```
Focus: Deep mastery, not exam prep
Add: Broader resources, projects, real-world application
Remove: Exam-specific tactics
Emphasis: Understanding over memorization, retention over speed
Examples: Language learning, musical instrument, programming mastery
```

**Variation 3: Professional Upskilling (Part-Time)**
```
Focus: Minimal viable study around work schedule
Add: Micro-learning (commute audio, lunch reading), weekend deep dives
Remove: Rigid daily schedules
Emphasis: Consistency over intensity, practical application
Examples: MBA courses, technical certifications, leadership training
```

## 💡 PRO TIPS

- **Test before you feel ready**: The struggle of retrieval strengthens memory
- **Teach to learn**: If you can explain it simply, you understand it
- **Sleep on it**: Consolidation happens during sleep — prioritize rest
- **Space, don't race**: 1 hour/day for 7 days beats 7 hours in one day
- **The "generation effect"**: Generate answers before seeing them
- **Interleave, don't block**: Mix topics to strengthen discrimination
- **Deliberate difficulty**: Easy studying = poor retention. Embrace productive struggle.

## 🔄 ITERATION WORKFLOW

1. **Diagnostic**: Test current knowledge, identify gaps
2. **Plan**: Build schedule using spacing and interleaving
3. **Execute**: Follow daily, track confidence ratings
4. **Weekly review**: Self-test, adjust focus areas
5. **Adapt**: If ahead/behind, adjust scope and priorities
6. **Practice exams**: Simulate conditions, analyze errors
7. **Final prep**: Light review, stress management, sleep
8. **Post-mortem**: What worked? Update for next learning goal
